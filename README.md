# CrazyCounter

Hello world
This is a simple counter app that demonstrates the use of the `useState` hook in React.

## Requirements

- Node.js
- npm

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Visit `http://localhost:3000/`
5. Enjoy!
